数据说明

本看板展示竞品抖音直播间的每日监控数据，由 GitHub Actions 每天北京时间 11:00 自动采集更新。

字段说明
  • 主推品：直播间重点推广的课程产品，含价格、上课时间、课程节数、赠品等
  • 话术策略：主播的核心转化逻辑和痛点设计
  • 用户画像：目标家长群体的特征描述
  • Highlights：值得关注的差异化亮点
  • 竞品动态：对比前后场次，发现产品方向和话术策略的变化
  • 竞争格局：各竞品整体策略对比和用户画像差异
数据来源：抖音直播间自动录制转写 · GitHub
